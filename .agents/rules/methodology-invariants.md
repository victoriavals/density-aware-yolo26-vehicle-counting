# Invarian Metodologis Terkodekan — Encoded Methodology Invariants

> **EN — TL;DR:** Hard methodology rules baked into the code that MUST NOT be relaxed: fixed seed 0 + identical config across all 8 variants (drop batch for ALL on OOM, never one); locked strata tiers (occlusion `0.10/0.35`, density `10/26`, COCO sizes) in `y26_strata.py`; Wilcoxon unit = AP per (class × stratum) with NO global row, 3 primary hypotheses uncorrected + secondary Holm in `y26_stats.py`; MAPE only for `y_t > 0`; pedestrians excluded from counting; density recomputed post-augmentation; group split before any training. Loosening any of these invalidates the ablation, the significance test, or the group-leakage guarantee — reviewers will flag it as a fatal defect.

Dokumen ini mendaftar invarian metodologis yang **sudah dikodekan** dalam skrip penelitian dan **dilarang dilonggarkan**. Setiap invarian berasal dari metodologi final BAB 3 dan aturan §12/§14 pada `CLAUDE.md` (SSOT). Melonggarkan salah satunya membatalkan salah satu jaminan inti tesis: keadilan *ablation study*, keabsahan uji signifikansi, atau jaminan bebas-kebocoran *group split*. Bila sebuah perubahan tampak membutuhkan pelanggaran, itu adalah keputusan besar — konfirmasi dulu ke Naufal (aturan §12.6), jangan diputuskan sendiri.

Konteks pendukung ada di [Framing kebaruan](../knowledge/thesis-framing.md), [Evaluasi terstratifikasi](../knowledge/evaluation.md), [Statistik](../knowledge/statistics.md), [DALW](../knowledge/dalw.md), [Penghitungan ByteTrack](../knowledge/counting.md), dan [Dataset & group split](../knowledge/dataset.md).

## Catatan penegakan

- **Konstanta tier ada satu sumber.** `SIZE_EDGES`, `OCC_EDGES`, `DEN_EDGES` (baris 36–38 `y26_strata.py`) menjadi *default* argumen di `gt_attributes`, `stratified_ap`, dan `occlusion_agreement`. Jangan meng-*override* argumen `occ_edges`/`den_edges`/`size_edges` dengan nilai lain saat menjalankan evaluasi produksi; parameterisasi itu hanya untuk uji unit dan analisis sensitivitas yang dilaporkan terpisah (mis. sensitivitas τ, bukan pergeseran ambang strata final).
- **Keluaran global bukan unit uji.** `stratified_ap` sengaja memancarkan baris `dim="global"` sebagai konteks; `paired_vectors` membuangnya secara default. Bila suatu skrip meminta `include_global=True`, itu penyimpangan dari protokol — perlu justifikasi eksplisit, bukan default.
- **Keluarga uji dipisah di satu tempat.** Seluruh logika utama-vs-sekunder + Holm terpusat di `run_wilcoxon_suite`. Jangan menyalin logika Wilcoxon ke skrip lain dengan aturan koreksi sendiri; panggil fungsi ini agar keluarga dan koreksi konsisten dengan BAB 3.11.4.
- **Aturan NEVER yang berkaitan.** Konfigurasi identik antar-varian menopang klaim *ablation*, tetapi ingat: HAM dan Lapisan P2 adalah **instrumen**, bukan kebaruan — invarian ini menjaga agar kontribusi tiap komponen terukur adil, bukan untuk mengangkat HAM/P2 sebagai temuan baru (lihat [Framing kebaruan](../knowledge/thesis-framing.md)).
